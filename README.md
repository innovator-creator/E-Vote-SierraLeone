# E-Vote SierraLeone
A Biometric and Transparent Digital Voting System for National Integrity

**By:** John Mark Fornah

## Overview
E-Vote SierraLeone is an AI-powered, blockchain-secured digital election system designed to eliminate voter fraud, underage registration, and multiple voting. It ensures that every vote is legitimate, every voter is verified, and every result is transparently recorded in real time.
