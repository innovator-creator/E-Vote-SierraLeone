## Overview

E-Vote Sierra Leone is an AI-powered, blockchain-secured digital election platform designed to modernize and protect the nation's voting process. The system ensures:

- No voter impersonation  
- No underage or invalid registrations  
- No multiple voting attempts  
- Every vote is valid, verified, and secured  
- Real-time transparent election results

By combining **biometric verification** with **blockchain transparency**, this solution makes elections tamper-proof, trustworthy, and fully auditable.
