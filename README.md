# E-Vote SierraLeone
*A Biometric and Transparent Digital Voting System for National Integrity*

**"One Nation. One Voter. One Verified Vote."**

**By:** John Mark Fornah

---

## Overview
E-Vote SierraLeone is an AI-powered, blockchain-secured digital election system designed to:

- Eliminate voter fraud  
- Prevent underage or invalid registrations  
- Stop multiple voting  
- Ensure every vote is legitimate, verified, and transparently recorded in real time  

This system combines **biometric verification** with **blockchain transparency**, ensuring **national elections are fair, secure, and accountable**.

---

## Key Features
- **Biometric Voter Verification:** Fingerprint, facial recognition, or palm scans  
- **Blockchain-Recorded Votes:** Immutable, transparent, and verifiable in real time  
- **Admin Dashboard:** Track voting status and monitor for anomalies  
- **Audit Trail:** Publicly verifiable results to prevent disputes  
- **AI Verification:** Automatically flags suspicious activities or duplicate attempts
